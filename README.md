# GO Smart Factory — Landing Page

Landing page de marketing do GO Smart Factory (MES). HTML + CSS + JS puro, sem build, sem dependências além de Google Fonts, EmailJS (envio do formulário) e Google Analytics (GA4).

## Deploy

Publicado via Vercel (import direto deste repositório, sem configuração de build — é site estático).

## Formulário de contato

Envio via [EmailJS](https://www.emailjs.com) — credenciais (Service ID, Template ID, Public Key) estão embutidas no `index.html` (chave pública, seguro expor no client-side por design do EmailJS).
